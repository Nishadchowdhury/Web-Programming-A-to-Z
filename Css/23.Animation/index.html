let's explore Animation

আমরা চাইলেই কোন এলিমেন্ট কে transtion: transition এর সাহায্যে এক স্থান থেকেও অন্য থানে সরাতে পারি , কিন্তু এটাকে যদি বার বার করতে চাই তাহলে Animation করতে হবে ।


structure of Animation

প্রথমে একটা keyframes নিতে হবে ।
তারপর একটা 2n bracket এর ভেতর from { first condition } এর ভেতর Animation এর প্রথম অবস্থা লিখতে হবে ।

তারপর to { second condition } এর ভেতর animation এর second condition লিখতে হবে । animation structure এর ২টা ধরন আছে

structure-1

 @keyframes sliding {
from{
background-color: lightblue;
etc }

to{
background-color: magenta;
etc }

}

structure-2

position:relative; করতে হবে class এর ।

animation কোন element এ apply করতে হলে সেই element এর class এর মধ্যে animation: proparty এর সাহায্যে apply করতে হয় ।

.class or #id {
animation: name duration timing-function delay iteration-count direction fill-mode;
}

name = ( keyframes এর সাথে যে animation এর নাম দিয়েছিলাম সেটা )
duration = কত সময় ধরে animation চলবে ।
timing-function = animation যে গতি পথে চলবে (ease-out )
delay = animation শুরু হওয়ার আগে কতক্ষণ দেরি করতে হবে ।
iteration-count = animation টা কয়বার চলবে । (infinite / n)
direction = animation এর দিক নির্দেশ করা ।
{ (reverse = পেছন/শেষ থেকে শুরু) , ( forwards/normal = শুরু থেকে শুরু ) , ( alternate = শুরু > শেষ > শুরু ) , (alternate-reverse = শেষ > শুরু > শেষ )}
fill-mode = (এই বিষয়-টা ক্লিয়ার হয় নাই । )
