>>> String Concatenation <<<

String Concatenation হল একটা String এর সাথে আন্য একটা String কে অন্য একটা string এর সাথে যোগ করা ।
( + ) বা Addition operator এর সাহায্যে String Concatenation করা হয় ।

যখন এই Addition operator টা string এর ক্ষেত্রে ব্যাবহার করব তখন সেটা String Concatenation এর কাজ করবে ।


২ টা string এর মধ্যে String Concatenation করার সময় এদের মধ্যে ফাঁকা বা space দিতে চাইলে এই নিয়মে করতে হবে

{( "string " + "string" ) বা ("string" + " " + "string")}
এখানে উভয় ক্ষেত্রে
result = string string

Ex:- let hi = "red" +" "+ "blue " + "Green" ;
console.log(hi);
result = redblue Greenbr

attention

javascript এ String Concatenation এর সময় কোন number String এর সাথে কোন number কে যোগ করলে বা Addition (+) করলে পুরো result কেই String বানিয়ে পেলবে ।
এটা শুদু Addition operator এর ক্ষেত্রে

কিন্তু subtraction বা অন্য সব operator এর ক্ষেত্রে string number কে javascript number হিসেবেই ধরবে
যেমন :-
let hi = 3 + "2";
result + 32 ।

আবার :-
let hi = 5 - "2" ;
result = 3 ।
