Switch

Switch অনেকটা if else , statement এর মত ।
যখন আমার কাছে এমন অনেকগুলা value থাকে তবে সেই value এর উপর ভিত্তি করে কাজ কর ।
যেমন ১ হলে এটা কর , ২ হলে ওটা কর , ৩ হলে সেটা কর - অথবা অন্যটা কর।

মোট কথা এটা if else এর একটা spacial / faster version ।


if else ,statement রান করলে সেখানে প্রতিটা লাইন execute হয় অর্থাৎ প্রতিটা লাইন আলাদা আলাদা করে পড়ে পড়ে যাচাই করে ।
যখন অনেকগুলা value কে যাচাই করে দেখার বিষয় আসে তখন if else এর , লাইন বাই লাইন execute করার কারনে অনেক সময় নস্ট হয়
তাই , বেশি পরিমানে value থাকলে সে সব ক্ষেত্রে switch case ব্যেবহার করতে হয় ।
