>>> comparison operator <<<

দুইটা value এর মধ্যে তুলনা করার ক্ষেত্রে comparison operator ব্যাবহার করা হয় ।
অন্যভাবে বলাযায় value এবং variable এর মধ্যে তুলনা করতে ও সম্পর্ক নির্ণয়ের ক্ষেত্রে ব্যেবহার করা যায় ।
comparison operator মুলত conditional statement গুলুতে বিভিন্ন value এর মধ্যে তুলনা করতে এবং ফলাফলের উপর ভিত্তি করে সিদ্দান্ত নিতে সাহায্য করে ।


comparison operator ছয় প্রকার ।
1. greater than ( > )
2. less than ( < )
3. greater than or equal ( >= )
4. less thanor equal ( <= )
5. equal equal ( == )
6. not equal ( != )

>>> greater than ( > ) <<<

greater than একটা ভ্যালু আর একটা ভ্যালুর থেকে বড় কিনা সেটা চেক করে ।

ex:-

result = true

if , 2 greater tahn 1 = true
or , 1 greater than 2 = false

>>> less than ( < ) <<<

less than একটা ভ্যালু আর একটা ভ্যালুর থেকে ছোটো কিনা সেটা চেক করে ।

ex:-

result = true

if , 2 greater than 5 = false
if , 2 less than 5 = true

>>> greater than or equal( >= ) <<<

এটা মুলত বা পাশের value এর তুলনায় ডান পাশের value ছোট কিংবা সমান কিনা সেটা চেক করার কাজে ব্যাবহার করা হয় ।

এখানে দুটা প্রশ্ন করা হয় ।

এখানের যেকোনো একটা প্রশ্নের উত্তর যদি true আসে তবে মুল result আসবে true ।

ex:-
