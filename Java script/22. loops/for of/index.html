for of

for of এর কাজ হল , এটার সাহায্যে আমরা iteration করতে পারি । এটার সাহায্যে iteratable object যেগুলা । যেমন array , string , map , set ইত্যাদি ।

loop এর iteration যত গুলা element আছে তত বার চলবে ।
এখানে প্রতিটা iteration এ name নামের array থেকে একটা একটা করে element এসে x এর মধ্যে store হবে ।
