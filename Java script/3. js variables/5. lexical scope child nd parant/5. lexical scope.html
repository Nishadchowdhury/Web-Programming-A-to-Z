lexical scope

এটিতে একটি প্যারেন্ট এবং এক বা একাদিক child function থাকতে পারে ।

সাধারনত কোন লোকাল টার্গেটেড ভ্যালু কে গ্লোবালি এসাইন করা জায়না ।
কিন্তু lexical scope এর ক্ষেত্রে প্যারেন্টের ভেরিএবেল child function গুলি ইউজ করতে পারবে ।

এবং প্যারেন্টের ভেরিএবেল গুলির ভ্যালু প্যারেন্টের child function এ আপডেট বা রিএসাইন করা যাবে ।
