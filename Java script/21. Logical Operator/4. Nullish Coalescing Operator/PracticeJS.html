মোট কথা 2 :-
কোন ভ্যেরিএবল set বা define করা আছে কিনা সেটা চেক করা হয় এই nullish বা null Coalecing operator এর সাহায্যে ,
আর যদি operator দেখে যে ভ্যেরিএভল টা null বা undefine অবস্থায় নেই তখনই সেই ভ্যেরিএবল টাকে সেট বোঝায় ,
set বা define থাকলে ভ্যেরিএবিলের ডাটা স্টোর হবে ।
আর সেট না থাকলে nullish operator এর ডাটা স্টোর হবে ।
