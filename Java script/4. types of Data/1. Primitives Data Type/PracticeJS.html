4. Null

যেমন যে ভ্যালুটা আমি জানি না কি হবে তখন সে ক্ষেত্রে সে ভেরিএবল এর ভ্যালু (Null) দিব , যাতে করে পরবর্তীতে সেটাকে এসাইন করা যায় ।

আবার ,,
Null মানে হল Nothing অর্থাৎ কিছুইনা । Null data কে একটা object হিসেবেও বিবেচনা করা হয় , কারন javascript এ Null কে object হিসেবে ধরা হয় ।

Error:-
javascript এর কোন error যদি Null হয় তবে বুজতে হবে যে ঐ ভেরিএবলে কোন ভ্যালু প্রবেশ করেনি ।

5. Undefined
javascript এ Undefined ও null এর মতই । এটার দ্বারা কন variable এ কিছু বিদ্যমান নেই বু্ঝায় । পার্থক্য হল javascript এ Null কে একটা object এবং Undefined কে একটা Data type হিসেবে ধরা হয় ।

অর্থাৎ কোন ভেরিএবল ডিক্লের করার পর যদি সেটা কোন প্রকার ভ্যালু দিয়ে এসাইন না করা হয় তবে সেটা বাই ডিফল্ট Undefined টাইপের হয় । >>>>>>>>{Output (Undefined)}

Error:-
javascript এর কোন error যদি Undefined হয় তবে বুজতে হবে ঐ ভেরিএবলে কোন ভ্যালু এসাইন করা হয়নি ।
