Primitives Data Type

Primitives Data Type হল যে সকল ডাটা টাইপের মধ্যে at a time শুধু মাত্র একটা টাইপের dataই রাখা যায় ।

Primitive data type সাধারনত ৬ প্রকার

1. Boolean
2. Number
3. String
4. Null
5.Undefined
6. Symbol

1. Boolean
Boolean সত্য এবং মিথ্যা represent করে ।
>>>>>> {Output (true,false) }

2. Number
javascript এ যখন শুধুমাত্র কোন number কে represent করতে হয় তখন number data type ব্যবহার করা হয় । নাম্বার কে Decimal(দশমিক) সহ ব্যবহার করা যায় ।
>>>>>>>>{Output (20 , 5.7)}


Number এর ২ টা ভ্যালু আছে । {Infinity , NaN} কোন Number কে 0 দিয়ে ভাগ করলে >>>>Infinity<<< দেখাবে ।
কোন "String" কে কোন Number দিয়ে ভাগ করলে >>>>NaN<<<< দেখাবে ।

3. String
Double Quote বা single Quote এর মধ্যে যা থাকে তাই String সেটা Number বা text যেকোন কিছু হতে পারে ।
>>>>>>>>{Output (Hello , Nishad , 20 , 5.7)}


সতর্কতা ঃ- " " এর মধ্যে থাকা কোন লেখার মাঝে যদি আবার কোন Double Quote আসে তখন সেই Double Quote ডান পাশে back slash (\) দিতে হবে

Single Quote এর খেত্রেও বেপারটা একই ।
>>>Ex:-
[let name = "red , it's the mame \" of a color";

console.log(name);

{Output >>>> (red , it's the mame " of a color)} ]

এর বাইরেও backtick ব্যাবহার করা যায় Qoute এর পরিবর্তে (``)
backtick এর ভেতরে ভেরিএবল এবং String একই সাথে রাখা যায় । এখেত্রে ভেরিএবল কে String এর ভেতর

result:-

এভাবে লিখতে হবে ।

এখানে js $ Symbol টাকে দেখা মাত্রই বুজবে যে এর পাশের second bracket এ কিছু আছে যা আমাকে execute করতে হবে ।

4. Null

যেমন যে ভ্যালুটা আমি জানি না কি হবে তখন সে ক্ষেত্রে সে ভেরিএবল এর ভ্যালু (Null) দিব , যাতে করে পরবর্তীতে সেটাকে এসাইন করা যায় ।

আবার ,,
Null মানে হল Nothing অর্থাৎ কিছুইনা । Null data কে একটা object হিসেবেও বিবেচনা করা হয় , কারন javascript এ Null কে object হিসেবে ধরা হয় ।

Error:-
javascript এর কোন error যদি Null হয় তবে বুজতে হবে যে ঐ ভেরিএবলে কোন ভ্যালু প্রবেশ করেনি ।

5. Undefined
javascript এ Undefined ও null এর মতই । এটার দ্বারা কন variable এ কিছু বিদ্যমান নেই বু্ঝায় । পার্থক্য হল javascript এ Null কে একটা object এবং Undefined কে একটা Data type হিসেবে ধরা হয় ।

অর্থাৎ কোন ভেরিএবল ডিক্লের করার পর যদি সেটা কোন প্রকার ভ্যালু দিয়ে এসাইন না করা হয় তবে সেটা বাই ডিফল্ট Undefined টাইপের হয় । >>>>>>>>{Output (Undefined)}

Error:-
javascript এর কোন error যদি Undefined হয় তবে বুজতে হবে ঐ ভেরিএবলে কোন ভ্যালু এসাইন করা হয়নি ।
