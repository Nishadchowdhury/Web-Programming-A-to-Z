use of "prompt"

prompt হচ্ছে একটা function এটা alert function এর মতই শো হয় ।
prompt এর মাধ্যমে ইউজার থেকে ডাটা নেয়া হয় ।

prompt() ফাংশনে দুটি প্যারামিটার দেয়া যায়। প্রথম প্যারামিটারে আপনি যে মেসেজ ইউজারকে দেখাতে চান সেটা আর দ্বিতীয় প্যারামিটারে ইনপুট বক্সে ।

prompt এর ইনপুট বক্সে কোন ইনিশিয়াল নাম বা লেখা দেখাতে হলে ।

ex:- Code Page's Line (14)
prompt এর ভ্যালু লেখা শেষ হলে কমা >>> , <<< দিয়ে exemple এর মত করে লিখতে হবে ।
