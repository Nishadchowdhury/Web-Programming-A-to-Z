এটাকে পাওয়া যায় devtool/application এর মধ্যে
getitem() ব্যেবহার করে Session Storage এবং Local Storage থেকে কোন নির্দিষ্ট মান retrun করা যায় ।
setitem() ব্যেবহার করে Session Storage এবং Local Storage এর মধ্যে কোন নির্দিষ্ট মান সেট করা যায় ।
যেমন setitem('keyNmae' , value);

তবে সেটা শুধু মাত্র একটা key যখন একটা value store করে ।
তাই যারা string আকারে আছে তাদের ব্যেবহার যোগ্য করতে হলে আগে তাদের keyName.JSON এ convert করতে হবে ।
তারপর সেটার মধ্যে data read , write করতে পারবে ।

read Write

এখানে normal value যদি set করতে চাই তবে setItem() দিয়ে সেট করা যায়
কিন্তু যদি কোন একটা key এর value হিসাবে object কে সেট করতে চাই তবে সেট করার আগে সে obejct টাকে stringfied করে ।
আবার কোন একটা key কে read করতে চাই এবং সেটার value যদি কোন obejct হয় তবে সেটাকে নিয়ে JSON.parse করে সেটাকে ব্যেবহার করতে পারবো কারন obejct গুলা stringfied অবস্থায় থাকে ।


____________ Session Storage ___________
browser জতখন থাকবে অর্থাৎ আমি জতখন use করতেছি ততক্ষণ থাকবে , বন্ধ করে দিলে চলে যাবে ।
আবার পরে আবার logIn করলে পরে আবার ব্যেবহার করতে পারবো ।

____________ Local Storage __________

এবং এটা reload করলেও থাকে ।
