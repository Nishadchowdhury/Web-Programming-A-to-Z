operator

+ , - , * , / এদেরকে বলা হয় operator .

প্রত্যেক operator ই কিছুনা কিছু প্রদান করে বা retrun করে ।

>>> unary operator <<<

যে সকল operator এর একটি মাত্র operand তাদের unary operator বলে ।

>>>Negation operator ( - ) <<<

>>> - <<< বিয়োগ কে Negation operator বলে ।

যেমন
Ex:- let firstNumber = -5
এটা একটা Negation operator ।

Negation operator এর operand একটা থাকায় এটাকে বলা হয় unary operator .


একই ভাবে Addition operator কেও কোন string type এর number কে number এ convert করতে গেলে ব্যাবহার করা যায় ।

Ex:- +"50" ; এখানে ,
Number("50") আর +"50" একই ভাবে convert এর কজ করবে ।

>>> Binary operator<<<

যে সকল operator সবসময় 2 টা operand নিয়ে কাজ করে তাদের binary operator বলে ।

>>> Subtraction operator ( - ) <<<

Subtraction operator দ্বারা ২ টা operand এর মধ্যে বিয়োগের কাজ করা হয় বা কোন operand কে ঋণাত্মক করা হয় ।
Ex:- { let hi = 10;
let hello = 5;
let ans = hi - hello ;
result is 5
}

>>> Addition operator ( + )<<<

Subtraction operator এর মতই এর কাজের নিয়ম Addition operator এর দ্বারা যোগের কাজ করা হয় ।
let ans = hi + hello ;
result is 15

>>> Multiplication operator ( * ) <<<

Subtraction operator ও Addition operator এর মতই এর কাজের নিয়ম
Muiltplication operator দ্বারা গুনের কাজ করা হয় ।
let ans = hi * hello ;
result is 50

>>> Division operator ( / ) <<<

Devition operator দ্বারা ভাগের কাজ করা হয় ।
Ex:- { let hi = 20;
let hello = 2;
let ans = hi / hello ;
result is 10
}

>>>Modulus (Remainder % ) operator<<<

Modulus operator এর দ্বারা ভাগ শেষ বের করার কাজে করা হয় ।

Ex:- { let hi = 20;
let hello = 2;
let ans = hi % hello ;
result is 0
}

>>> Exponentiation operator ( ** ) <<<

Exponentiation operator এর সাহায্যে বর্গের কাজ করা হয় ।
Ex:- { let hi = 3;
let hello = 2;
let ans = hi ** hello ;
result is 9
}
