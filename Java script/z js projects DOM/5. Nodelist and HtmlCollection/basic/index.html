Nodelist

DOM এর ভাষায় এইচটিএমএল এর মধ্যে থাকা সবই আলাদা আলাদা node . কিছু আছে Element আবার কিছু আছে None Element , node


যেমন comment ও এক প্রকার node ,, এবং যেকোনো tag এর মধ্যে থাকা টেক্সট ও node এর অন্তর্ভুক্ত ।



NodeList কোন array না , তবে array এর মত দেখতে ।

যেমন কোন নরমাল array মানে হল ।

const theArray = [2, 3, 4, 5, 6, 10];
এখন আমি চাইলে এই array এর length বের করত পারি ।
চাইলে কিছু push() করতে পারি । এবং pop করতে পারি

কিন্তু Nodelist এর মধ্যে কিছু pop বা push করা যায়না ।
