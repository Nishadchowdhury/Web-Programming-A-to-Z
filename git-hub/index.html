গিট রিপজিটরি কি? (What is git repository?)

সহজ ভাষায় বলতে গেলে গিট রিপজিটরি হলো একটি ফোল্ডার বা ডিরক্টরি যেখানে প্রজেক্ট রাখা হয়। এটা আপনার লোকাল কম্পিউটারও হতে পারে বা কোন অনলাইন হোস্টিংও হতে পারে। এই রিপোজিটরি আপনার সম্পুর্ন কোডবেজ এবং প্রত্যেকটি রিভিশন হিস্টরি সেভ করে রাখবে।

git এর মদ্যে জেভাবে ফাইল অ্যাড করতে হয় ।

Git branch


git brach মুলত ব্যেবিহার হয় যখন অনেক জন developer একসাথে একটা প্রোজেক্ট নিয়ে কাজ করে ।
সবাই তাদের মত করে তাদের কাজ করবে তারপর সবার কাজ sync করে প্রোজেক্ট সম্পন্ন করা হয় ।

branch commands

  1. git branch = git branch এর মধ্যে থাকা branch এর list চেক করা ।
  2. git branch branch-name = নতুন একটা branch তৈরি করাতে (no-space) Prb=1 ।
  3. git branch -d branch-name = কোনো branch কে ডিলিট করতে (no-space) ।
  4. git checkout branch-name = এক branch থেকে অন্য branch এ Switch করাতে (no-space)
  5. git merge branch-name = অনেক গুলা branch কে
    এক সাথ করে একটা mein বা defult branch এর মধ্যে একাত্র করাকে বলে । pro=2
